# TestMail Hub 计划书实现对照表

更新日期：2026-03-31

这份文档用于把 [产品与研发计划书](./产品与研发计划书.md) 里的阶段目标，映射到当前仓库的真实实现状态，方便发版、对外说明和后续排期。

## 状态定义

- `已完成`：阶段核心目标和主要验收点已经落地
- `大部分完成`：主干能力已完成，但仍有少量缺口
- `部分完成`：已经有基础能力，但离计划目标还有明显差距
- `未开始`：当前仓库里还没有对应闭环

| Phase 1 | 验证码与链接提取升级 | 大部分完成 | 提取、分类、平台识别已落地，准确率优化仍持续进行 |

  - 成员列表已显示最近变更时间、操作人和变更动作
  - 成员页可直接打开“最近变更记录”抽屉查看该成员的治理留痕
- 平台级读取接口已补权限收紧：
  - 管理员列表已按角色与项目范围返回
  - API Token 列表仅可写管理员可读

- 审计日志页第二轮增强已落地：
